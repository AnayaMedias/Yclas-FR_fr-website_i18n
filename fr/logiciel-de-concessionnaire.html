 Pour tous ceux qui veulent créer une plate-forme de vente en ligne pour les voitures, Yclas est l'outil parfait. Comme Yclas met l'accent sur une expérience conviviale, il est conçu pour une utilisation simple et adapté à tout le monde. Aucune connaissance sur le développement d'un site web est nécessaire. En quelques minutes, vous pouvez créer votre propre site web de petites annonces en utilisant l'un des thèmes prédéfinis afin de créer un site immobilier. Notre script est livré avec toutes les fonctionnalités standard. Vous pouvez également par la suite, créer des fonctionnalités personnalisées directement liées aux voitures disponibles.

À qui cela s'adresse-t-il ?

Concessionnaires automobiles

Êtes-vous un concessionnaire de voiture à la recherche de possibilités pour développer votre entreprise? Voulez-vous mettre votre business en ligne, mais vous n'avez aucune connaissance à la programmation de sites Web? Dans ce cas Yclas peut être la solution parfaite pour vous. Avec Yclas vous pouvez facilement créer un site web de petites annonces et de présenter votre catalogue complet en ligne.

 Les annonceurs en ligne

Les annonceurs en ligne voulant créer une plate-forme de vente en ligne pour voitures ont de grandes possibilités en utilisant Yclas. Il est très facile de commencer et les thèmes personnalisés pour les voitures vous permettent de créer un site web de petites annonces de voiture en un clin d'œil. Si par exemple vous voulez donner aux clients la chance faire de la publicité de leurs voitures sur un site de voiture d'occasion locale, Yclas est un excellent script qui vous vous conviendra.

Caractéristiques (général)

Avec Yclas vous aurez la disponibilité d'utiliser toutes les fonctionnalités nécessaires à la création d'un site d'annonces avec succès. Les thèmes gratuits vous fournira des fonctionnalités standard comme la création de pages, créer des catégories, le téléchargement de photos, l'attribution des emplacements, de recherche, d'un riche éditeur HTML, une confirmation de Captcha, etc. À côté de cela, il ya la possibilité d'acheter des thèmes premium qui offrent des fonctionnalités supplémentaires comme s'inscrire aux services en utilisant ses comptes de réseaux sociaux, une conception adaptée et enfin avoir un soutien professionnel.

Des champs personnalisés (spécifiques pour le type de modèle)

  • Pour la vente par (liste) 
    Vendu par propriétaire ou le concessionnaire
  • Type d'annonce (de la liste) 
    Vendre une voiture ou à la recherche d'une
  • Faire (liste)
    Sélectionnez la marque à partir d'une liste
  • Autre marque (en plein champ)
    Si la marque ne figure pas
  • Modèle (champ libre)
    Nommez le modèle
  • Année (champ libre)
    Date de l'année
  • Kilomètres (champ libre)
    Nommez le nombre de kilomètres
  • Type de carrosserie (liste)
    Sélectionnez le type de carrosserie  dans la liste
  • Transmission (liste)
    Sélectionnez le type de transmission dans la liste
  • Système de transmission (liste)
    Sélectionnez le type d'une liste de transmission
  • Couleur (liste)
    Sélectionnez la couleur à partir d'une liste
  • Type de carburant (de la liste) 
    Sélectionnez le type de carburant dans la liste
  • Type (liste)
    Sélectionnez si la voiture est utilisée, prendre plus de bail, nouvelle ou endommagé
