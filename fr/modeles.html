CZ sale

The CZSale theme is a fully responsive theme that was developed with the Bootstrap framework. This makes the theme great for those that want to make their classifieds website available for both mobile devices and desktop computers or laptops. The responsive design will give the user the optimal user experience on every platform. Included are 7 different color schemes that allow you to adapt the looks. Infinite scroll, right-to-left and all other premium features are supported.
